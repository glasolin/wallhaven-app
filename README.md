# WallHaven App
Приложение предназначено для поиска картинок для фона с сайта [WallHaven](https://wallhaven.cc) при помощи [API](https://wallhaven.cc/help/api). Приложение должно состоять из следующих экранов:
1. Основной экран
2. Фильтры
3. Избранное
4. Настройки
5. Просмотр картинки

## Основной экран

Основной экран должен состоять из следующих элементов:

1. Плашка с названием
   * Кнопка перезагрузки картинок
   * Переход на сайт [WalHaven](https://wallhaven.cc)
2. Строка поиска по тегам
3. Кнопка выбора фильтров
4. Кнопка сортировки
5. Грид с картинками
6. Плашка навигации: Основной, Избранное, Настройки

## Фильтры
Экран фильтров должен состоять из следующих элементов:

1. Плашка с название экрана
   * Кнопка перехода на основной экран
2. Категория
3. Чистота
4. Отношение стороон
5. Разрешение (x, y)
6. Цвет
7. Выбор тегов
8. Сбросить фильтр

## Избранное

Экран избранного должен состоять из следующих элементов:

1. Плашка с названием
    * Переход на сайт [WalHaven](https://wallhaven.cc)
2. Грид с картинками
   * Кнопка удаления из избранного у каждой картинки
3. Очистка избранного
4. Плашка навигации: Основной, Избранное, Настройки

## Настройки

Экран настроек должен состоять из следующих элементов:

1. Плашка с названием
    * Переход на сайт [WalHaven](https://wallhaven.cc)
2. Поле с вводом API Key
3. Выбор темы (Системная, Светлая, Темная)
4. Использование хранилища
   * Круговая диаграмма
   * Занято избранным
   * Занято пол кэш
   * Свободно
5. Очистка кэша
6. Плашка навигации: Основной, Избранное, Настройки

## Просмотр картинки

Экран настроек должен состоять из следующих элементов:
1. Плашка с названием
    * Переход на страницу картинки на сайте [WalHaven](https://wallhaven.cc)
    * Разрешение картинки
    * Переход на основной эран или в избранное (в зависимости от того, откуда выбрана картинка)
2. Список плашек с цветами картинки
3. Список тегов
4. Таблица свойств
   * ID
   * Имя автора
   * Размер файла
   * Категория
   * Чистота
   * Разрешение
5. Добавление в избранное
